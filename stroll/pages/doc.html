配置说明

config.ini

  • 全局配置(global)
[global]
mvn_path=/data/www/tools/apache-maven-3.5.3/bin/mvn
dst_path=/home/develop/.jenkins/workspace
container_root_path=/data/www
local_share_path=/data/www/projects/stroll
producer_try_times=20
customer_try_times=80
属性 必须 默认值 描述
mvn_path Y 无 maven路径
dst_path Y 无 资源存放路径
container_root_path Y /data/www 容器内根路径
local_share_path Y 无 宿主主机与容器挂载路径
producer_try_times N 20 dubbo尝试次数
customer_try_times N 80 web尝试次数
  • jenkins配置(jenkins)
[jenkins]
config_xml_path=/home/develop/.jenkins/config.xml
job_dst_path=/home/develop/.jenkins/jobs
属性 必须 默认值 描述
config_xml_path Y 无 jenkins全局配置路径
job_dst_path Y 无 jenkins任务列表路径
  • 分支配置
[node]
branch_name=master
src_path=/data/www/projects/finance-hkjf
env_path=/data/www/projects/env_hk
producer_list=user,loan,payment,invest,secondary
producer_protocol_port_start=6000
producer_debug_port_start=5000
customer_list=management,financial,api,bi
customer_port_start=8000
customer_debug_port_start=7000
proxy_tomcat_port=8500
proxy_ssh_port=8522
proxy_zk_port=8521
backup_suffix=_backup
producer_prefix=finance-
producer_suffix=-service
customer_prefix=hk-
customer_suffix=-services
version=-1.0-SNAPSHOT
debug=1
agent=0
agent_ip=127.0.0.1:11800
属性 必须 默认值 描述
branch_name N master 分支名称
src_path Y 无 源码路径
env_path Y 无 测试环境资源路径
producer_list Y 无 生产者服务列表
producer_protocol_port_start N 6000 生产者dubbo协议端口
producer_debug_port_start N 5000 生产者debug端口
customer_list Y 无 消费者(web)服务列表
customer_port_start N 8000 消费者(web)服务端口
customer_debug_port_start N 7000 消费者(web)服务debug端口
proxy_tomcat_port Y 无 宿主主机映射docker容器的web端口
proxy_ssh_port Y 无 宿主主机映射docker容器的ssh端口
proxy_zk_port Y 无 宿主主机映射docker容器的zk端口
backup_suffix N _backup docker容器内备份服务后缀
producer_prefix N finance- 生产者资源前缀
producer_suffix N -service 生产者资源后缀
customer_prefix N hk- 消费者资源前缀
customer_suffix N -services 消费者资源后缀
version N -1.0-SNAPSHOT 资源版本号
debug N 1 服务debug开关
agent N 0 服务APM开关
agent_ip Y 无 APM集控地址
